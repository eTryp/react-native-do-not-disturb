# react-native-do-not-disturb
Enable and disable DND mode on Android devices.
## Installation

```sh
npm install react-native-do-not-disturb
```

## Usage

```js
import { multiply } from "react-native-do-not-disturb";

// ...

const result = await multiply(3, 7);
```
